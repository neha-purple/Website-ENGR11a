Project 1: Introduction to 3D Printing

Part 1: Nametag

I would consider this print successful, although there are noticeable flaws. It came out legible which is the most important element, but there were noticeable flaws. Mainly I had problems with some stringing that I had to clip off afterward. Its not the cleanest result but it still serves its purpose. I think if i scaled it up a little then this problem would not be a concern. If I did this again I would also slow down the print speed for cleaner edges, and polish or paint the surface for a more clean appearance.

Part 2: Calibration (“torture”) print

I believe this print went very well, and I'm satisfied enough with the final outcome that it is by far my favorite. I really love the "furry" texture that came out around its ears! I know that these "mini strings" are technically not supposed to be there, but I think they are a great unintended detail. Also, this "webbing" is so thin that it would be able to be brushed off easily if I did not like it. The details turned out really nicely, and although there is not as much definition on the face as some people might like, I think that this was the intended result of the design and the little details on the face are still clear enough to effectively communicate the features of the cat. I was especially impressed by how well the printer handled the overhangs. The bridges were a little rough, but not to the point of failure. And in terms of testing the printer’s capabilities, I think this was a successful result.

Part 3: Open-source assistive devices

This print had more challenges. It is supposed to be a light switch extender that can help people with mobility issues (arthritis, limited grip strength, finger dexterity) and other disabilities to use the light switch. Unfortunately, some of the supports collapsed inside during printing, leaving completely unusable. When I first saw it, I thought I could pull it out, but it is too stuck to easily pop off (and I no longer had the clippers to help with this). Surprisingly this blockage was only at the start of the hole and the deeper part is clear, so maybe if I can still get it out someone then it could work. In that scenario I would still have some problems because the opening is still smaller that the average light switches I see and I would have to do some trial and error to make it compatible with a switch. The finish, especially on the top, looked very “3D printed” and I wished there were ways to make it smoother; maybe sanding or coating would help. On the bright side... it’s easy to locate in the dark and works as a good prototype. For a decorative or more permanent version, I would obviously hope for larger, and more polished options. I do think this print is a failure because the main functionality is compromised. But, if I were to print it again, I would try to make it slightly larger and with supports that could detach easier.
